# Hyper Local Delivery Dispatcher - Frontend

Frontend application for the Hyper Local Delivery Dispatcher project.

## Tech Stack

- React
- Redux Toolkit
- React Router DOM
- Axios
- Tailwind CSS
- Socket.IO Client
- Vite

---

## Features

### Authentication

- Login
- Register
- Logout

### Customer Dashboard

- Create Order
- View Orders
- Track Status

### Admin Dashboard

- View Orders
- Assign Delivery Partners
- Monitor Deliveries

### Delivery Dashboard

- Accept Delivery
- Complete Delivery
- Manage Assigned Orders
