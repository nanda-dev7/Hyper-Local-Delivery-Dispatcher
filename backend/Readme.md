# Hyper Local Delivery Dispatcher - Backend

Backend API for the Hyper Local Delivery Dispatcher project.

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcryptjs
- Socket.IO
- REST Client

---

## Features

### Authentication

- Register User
- Login User
- Current User Profile
- JWT Authentication

### Role Based Access Control

- Customer
- Delivery Partner
- Admin

### Order Management

- Create Order
- Get Orders
- Get Single Order
- Update Order Status

### Delivery Management

- Assign Delivery Partner
- Accept Delivery
- Complete Delivery
- Track Delivery Status
